About This App

Welcome to our Image Processing App! This web application allows users to perform advanced image processing operations seamlessly. It is designed for ease of use, offering a variety of tools for image manipulation, including image stitching, edge detection, and AI-based human detection.

Features

  • Image Stitching: Upload multiple images and stitch them together to create a single, coherent image. This feature is perfect for panorama creation or merging related images.
  • Edge Detection: Apply Canny edge detection or Difference of Gaussians (DoG) to the stitched image. Use the sliders to dynamically adjust kernel sizes for better control over the edge detection process.
  • Morphological Operations: Fine-tune your edge detection with the ability to adjust the kernel size for morphological operations, allowing you to smooth or highlight edges as needed.
  • AI-Based Human Detection: Detect human figures in images using pre-trained YOLOv3 models with a confidence threshold of 50% or more, ensuring accurate detection results.
