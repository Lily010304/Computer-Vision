How to Use Image Stitching and Morphological Operations

Follow these steps to stitch multiple images and apply morphological operations:

  1. Click on "Browse" to upload multiple images for stitching (hold CTRL or SHIFT to select more than one image).
  2. Click "Upload Images" to start the stitching process.
  3. Once the images are stitched, you'll be able to adjust the size of the morphological operation using the provided slider.
  4. After adjusting the slider to your desired kernel size, the system will apply the morphological operation to the stitched image.
