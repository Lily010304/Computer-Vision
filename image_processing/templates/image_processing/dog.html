How to Use the Difference of Gaussians (DoG)

Follow these steps to apply the DoG operation on your image:

  1. Click on the "Browse" button to upload your image.
  2. Select the first kernel size in the Mask Size 1.
  3. Select the second kernel size in the Mask Size 2.
  4. Click "Apply DoG" to process the image.
  5. The image with the DoG effect applied will be displayed shortly.

You can tweak the kernel size to modify the effect of the smoothing.
